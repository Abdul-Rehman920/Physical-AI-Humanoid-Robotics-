# RAG Retrieval Pipeline

## Usage

### Run Test Queries
```bash
uv run retrieval.py
```

### Single Query
```bash
uv run retrieval.py --query "What is Physical AI?"
```

### View All Data
```bash
uv run retrieval.py --all
```
